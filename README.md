# 🚀 TSBS Analytics

> 时间序列基准测试数据分析平台

### 🔐 启动
```bash
# 前台启动（可看到实时日志）
./start.sh

# 后台启动（不占用终端）
./start.sh -d -o
```

### 📊 管理命令
```bash
./status.sh   # 查看运行状态
./stop.sh     # 停止服务
./restart.sh  # 重启服务
```

### 📱 访问应用
- **登录页面**: http://localhost:5001/login.html
- **主界面**: http://localhost:5001/ (需要登录)
- **基准配置**: http://localhost:5001/baseline (需要登录)
